# 前置

## 动态引入

```
<link href="https://unpkg.com/@ionic/core@1.7.16/css/ionic.bundle.css" rel="stylesheet">
<script src="https://unpkg.com/@ionic/core@1.7.16/dist/ionic.js"></script>
```

## 常用命令

```
//ionic下载项目模版

//在项目要存放的位置打开命令行
//空白模版
$ ionic start myAppName blank
//带有标签页的模版（默认模版）
$ ionic start myAppName tabs
//带有左侧边栏的模版
$ ionic start myAppName sidemenu

// 为项目添加平台支持

//在项目目录中打开命令行
//添加Android平台支持
$ ionic platform add android
//添加IOS平台支持
$ ionic platform add ios

// 将项目打包成apk

//在项目目录中打开命令行
$ ionic build android
```
