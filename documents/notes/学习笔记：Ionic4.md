# 学习笔记

## 引入图标的另一种方式

```
<!--引入标准图标-->
<ion-icon name="add-circle-outline"></ion-icon>
<!--引入自定义图标-->
<ion-icon src="/path/to/external/file.svg"></ion-icon>
<!--区分平台-->
<ion-icon ios="ios-heart" md="md-heart"></ion-icon>
<!--设置图标大小-->
<ion-icon size="small"></ion-icon>
<ion-icon size="large"></ion-icon>
```
